About

 

{{.Layout.Platform}} is a simple link aggregation and discussion platform.

 

Link and news sharing

Users can share useful and interesting links and start a discussion. If you have an account, use the link here to submit a link and start a discussion about it.

 

Discussions and Comments

Anyone on {{.Layout.Platform}} can write a comment to any post shared and others can reply to that comment as well. In this way users will start a discussion and engage on any kind of post.

 

Upvote & Downvote

If a user founds a useful post shared on the platform, he/she can show their appreciation by up-voting that post. Not every post may be welcomed by your community. If this is the case, users can react to that post by down-voting it.

 

Works with invitation

Not everyone can join {{.Layout.Platform}}. Users can only be signed up to the platform if they are invited. If you would like to have an account on {{.Layout.Platform}} you can find someone who already has an account and ask for an invitation.

 

Ask & Show

You can submit a question about any topic to the community and start collecting feedback. E.g. You just finished a project and you are desperately in the need of feedback? We have all been there. Just post your project to the community and collect all needed feedback so you can continue improving it.
